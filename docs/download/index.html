Download one small installer. It fetches the official Mix Studio repository, keeps your gallery local, and connects to the ComfyUI setup you already use.

Download for Windows Windows 10/11 · Existing ComfyUI installation
01

Download

Save install.bat from this page. Your browser may ask you to confirm keeping an executable script.

02

Run

The downloader installs Git through Windows Package Manager if needed, then clones Mix Studio into your user folder.

03

Connect

The Mix Studio setup window opens automatically so you can select your ComfyUI URL, folders, and model families.

Transparent by design. The downloader retrieves only the public repository at BlackMixture/KreaStudio. It does not install ComfyUI or model files automatically.
