Quick start

Set up once. Create from your phone.

The Windows installer prepares the generation machine. Tailscale turns its Mix Studio address into a private studio you can carry.

Prepare the machine

Use Windows 10 or 11 with an NVIDIA GPU, current drivers, and enough SSD space. The wizard detects GPU memory and system RAM, then recommends model families that fit the machine.

Download and run

Save install.bat from this page and open it. Windows may ask you to confirm keeping or running an executable script.

Follow the setup window

Choose a new official ComfyUI Desktop installation or an existing environment. Borderline workflows show an offload warning, difficult models stay available as an informed opt-in, and model files already registered through ComfyUI are reused. A manual models-folder picker remains available.

Connect both devices

Install Tailscale on the Windows desktop and your phone, then sign in to the same private network.

Open Mix Studio on your phone

Start Mix Studio on the desktop and open the labeled Tailscale: URL shown in its terminal. Add it to the phone's home screen for an app-like launch.

Transparent by design. The bootstrap retrieves the public BlackMixture/Mix-Studio repository, the signed official ComfyUI Desktop installer, and the curated model or node sources selected in the wizard. Mix Studio applies its optimized workflow settings in the app. Existing settings and hardware choices are preserved, and the included uninstaller leaves ComfyUI and shared models in place.
